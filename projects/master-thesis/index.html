Project 01 · Master's thesis

How does DNA sequence affect cytosine deamination?

Computational chemistry project investigating how neighboring DNA bases influence cytosine and 5-methylcytosine deamination using MED interaction energies and DTSS analysis.

MED DTSS CAMM Das dispersion A-DNA vs B-DNA
Open README View scripts
Methods

MED, CAMM, Das dispersion, DTSS

Tools

Python, GAMESS, X3DNA DSSR, Git, Linux

Focus

DNA sequence context, A-DNA vs B-DNA, C vs 5mC

Workflow

  1. Select substrate and transition-state models from literature.
  2. Generate A-DNA and B-DNA structural models.
  3. Create single-nucleotide variants around the reactive cytosine.
  4. Calculate CAMM electrostatics and Das dispersion components.
  5. Combine components into MED interaction energies.
  6. Calculate DTSS values and visualize sequence-dependent trends.
