# Models

This folder contains the trained DeepMD-kit models for the formic acid dimer machine learning interatomic potential.

```text
models/
├── graph.pb
└── graph-compress.pb
```

## Model files

| File | Description | Use |
|---|---|---|
| `graph.pb` | Frozen DeepMD model generated after training | Used as the standard trained model file |
| `graph-compress.pb` | Compressed DeepMD model generated from `graph.pb` | Used for more efficient molecular dynamics simulations in LAMMPS |

## Model generation

The model was trained with DeepMD-kit using the 500 K formic acid dimer trajectory as the training dataset and the 298 K trajectory as the validation and testing dataset.

After training, the model was frozen:

```bash
dp freeze -o graph.pb
```

Then the frozen model was compressed:

```bash
dp compress -i graph.pb -o graph-compress.pb
```

The compressed model was later used in LAMMPS molecular dynamics simulations.

## Usage in LAMMPS

The compressed model can be used in a LAMMPS input file with the DeepMD pair style:

```lammps
pair_style deepmd ../models/graph-compress.pb
pair_coeff
```

If the LAMMPS input file is located directly inside the `lammps/` folder, the relative path should point to the model file in the `models/` folder:

```text
../models/graph-compress.pb
```

## Notes

The files in this folder are binary model files and should not be edited manually.

If the model files are large, they may be stored using Git LFS or provided through a GitHub Release.

If the repository uses Git LFS, download the model files after cloning with:

```bash
git lfs install
git lfs pull
```

## Reproducibility

These model files are part of the final workflow:

```text
DeepMD training
        ↓
Model freezing
        ↓
Model compression
        ↓
LAMMPS molecular dynamics simulation
```

The model can be regenerated by running the project workflow from the original DeepMD-formatted datasets stored in the `data/deepmd/` folder.
